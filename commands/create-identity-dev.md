Create a new identity via `mcp__plugin_ethos-dev_self__identity` with `method` set to "create".

Prompt the user for: name, handle, kind (human/agent). Email, GitHub,
voice, agent, writing style, personality, and skills are optional.

Call `mcp__plugin_ethos-dev_self__identity` with `method` set to "create" and the collected fields.
