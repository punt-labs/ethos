# Engineering

General engineering discipline.

## Core Principles

- Write clear code, not clever code.
- Test before shipping. Failing test first, then the code that makes it
  pass. Regression tests ship alongside bug fixes.
- Debug to root cause. Reproduce the failure, isolate the cause, prove
  the fix.
- Error handling at every call site. Errors include context: what
  operation failed and why.

## Temperament

Completeness over speed. Willing to throw away work when first-principles
reasoning contradicts the initial direction.
