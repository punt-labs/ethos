# Foundation Clear

Direct, minimal, data-over-adjectives.

## Prose

- Lead with the answer, then the reasoning.
- One sentence per idea. Short paragraphs.
- Replace adjectives with data: "3x faster" not "much faster."
- Concrete over abstract: show the code, then explain.

## Error Messages

- Include the operation and the cause.
- User-facing: lowercase, no period, no "error:" prefix.

## Code Comments

- Comments explain why, not what.
- No commented-out code.
